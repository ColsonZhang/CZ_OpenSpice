About The Project

CZ_OpenSpice is an online circuit simulation platform based on the core of the open source circuit simulator. It can provide integrated circuit developers and designers with a friendly interface, easy to use, and complete circuit design environment, including circuit schematic drawing, multiple modes of circuit simulation, Multi-interface result rendering and other functions enable developers to complete a relatively complete design process on the platform.

The simulation platform is deployed on the server and is opened and used by users in the form of web, eliminating the user's download and installation process, and at the same time, it can also enable multiple platforms and multiple users to use in parallel.

CZ_OpenSpice is an open source project, and the code is open sourced on the Github platform and Gitee platform. This project uses a variety of open source technologies. The front-end drawing circuit schematic diagram mainly uses the open source js library mxgraph; the back-end execution circuit simulation mainly uses the PySpice library; the entire server uses the Tornado library to deploy.

This project is a graduate design project aimed at building a flexible and easy-to-use circuit simulation platform. This project is for learning only; in addition, the platform will collect user design information with the knowledge of users. Please do not develop private projects on this platform.
