# CZ_OpenSpice

# 项目简介

本项目旨在搭建一个web版本的spice电路仿真平台。当前项目仍处于不断更新中。

# 项目框架

## MyJade

能够绘制电路原理图的web前端代码，主要参考MIT的Jade项目。

## Server

整个项目的后端服务器代码，主要基于Tornado、Bokeh框架开发。
